# COVID-19

概述国内外疫情，分别就疫苗接种情况和数据进行可视化设计

# 快速上手

## 首次启动请安装依赖

```
npm install
```

## 启动

```
npm run start
```

## 打包

```
npm run build
```

# 规范

## 分支类型

main: 主分支

dev: 研发分支

feat: 特性

fix: 修复

hotfix: 紧急修复

docs: 文档分支

## 分支命名

分支创建请遵循： `分支类型/名称`.

例如：

`feat/overview` 

## 代码合并流程

新建一个feat/fix/docs分支，请push到仓库后，手动创建PullRequest，然后合并进dev分支，切勿直接操作main、dev分支。
